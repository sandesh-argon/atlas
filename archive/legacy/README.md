# Legacy Lineage Archive

This archive preserves filtered historical code/docs lineage from:

- v1
- v2.0
- v2.1
- v3.0
- v3.1

## Inclusion Policy

Included:

- code
- scripts
- docs
- configuration files

Excluded:

- data directories (`Data`, `data`, `outputs`, `models`)
- virtual environments and binary artifacts
- node_modules and build outputs
- files larger than 10MB

Use this archive for methodological lineage review, not for data reproduction.
