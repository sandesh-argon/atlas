# Atlas Sample Dataset

This directory contains a tiny deterministic sample to validate setup and smoke-test pipeline wiring without the full Zenodo dataset.

## Scope

- 5 countries
- 5 years (2020–2024)
- 50 indicators
- Long-format panel rows

## Files

- `sample_panel_long.csv`: minimal panel-style dataset
- `sample_metadata.json`: sample schema and generation details

## Notes

- This sample is **not** representative of full Atlas statistical properties.
- Use it only for quick integration tests and tutorial walkthroughs.
- Full research reproduction requires Zenodo-hosted artifacts.
