# Atlas Pipeline

Pipeline code organized by stage:

- `a2_granger/` — temporal prefiltering and FDR correction
- `a3_structure/` — PC-Stable structure pruning and cycle removal
- `a4_bootstrap/` — bootstrap effect estimation and validation
- `shap_analysis/` — SHAP and QoL calibration preprocessing
- `findings_extraction/` — findings ranking/extraction utilities

Legacy lineage scripts from v2.1 are included for reproducibility.
